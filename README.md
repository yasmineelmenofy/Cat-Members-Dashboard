# Member Management System

A full-stack dashboard application built with **Node.js, Express, MongoDB, and EJS** using MVC architecture.  
The system allows managing club members with full CRUD functionality.

---

## 🚀 Features

- ➕ Add new members
- 📋 View all members in a styled dashboard
- ✏️ Edit member information
- 🗑️ Delete members with confirmation
- 🔎 Filter members
- 🎨 Clean and responsive UI
- 🔐 Environment variables for secure configuration
- 🧠 MVC architecture structure

---

## 🛠️ Technologies Used

- **Node.js**
- **Express.js**
- **MongoDB**
- **Mongoose**
- **EJS**
- **CSS**
- **Method-Override**
- **Dotenv**

This project follows the **MVC pattern**:

- **Model** → Handles database logic
- **View** → EJS templates
- **Controller** → Business logic
- **Routes** → Handle HTTP requests
